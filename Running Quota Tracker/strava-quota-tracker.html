Connect to Strava

You need a free Strava API app to pull your data. Takes about 2 minutes.

  1. Go to strava.com/settings/api
  2. Create an app — name it anything (e.g. "My Tracker")
  3. Set Authorization Callback Domain to localhost
  4. Copy your Client ID and Client Secret below

After authorizing, Strava redirects to localhost (which won't load — that's fine). Copy the code=... value from the URL bar and paste it below.
